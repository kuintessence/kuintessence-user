## 介绍

坤仪万象是一个为高性能计算设计的强大流程和集群管理软件，旨在简化和优化 HPC 任务的提交、调度和管理。

## 特性

- **自动化任务调度**：依据资源和优先级智能调度任务。
- **用户友好的界面**：轻松提交、监视和管理任务。
- **优化资源利用**：确保 HPC 资源的最大化使用。
- **安全性**：提供用户验证和权限管理，确保集群的安全。
- **可扩展性**：支持多种集群管理软件，如 SLURM、PBS、SGE 等。
- **接入简单**：无需修改集群配置，只需简单配置即可接入。
